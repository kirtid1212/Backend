# Public Assets

This directory contains public assets for the web application.

## Notification Icons

You need to add the following icon files for push notifications:

- `icon-192x192.png` - Main notification icon (192x192 pixels)
- `badge-72x72.png` - Badge icon for notifications (72x72 pixels)

### How to create icons:

1. **icon-192x192.png**: Create a 192x192px PNG image with your app logo
2. **badge-72x72.png**: Create a 72x72px PNG image with a simple badge/icon

You can use tools like:
- Canva
- Figma
- Adobe Photoshop
- Online icon generators

### Recommended design:
- Use your brand colors
- Keep it simple and recognizable
- Ensure good contrast for visibility
- Use transparent background for icon-192x192.png
